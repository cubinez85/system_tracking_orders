📋 Промт проекта "Система трекинга заказов" 
Описание проекта
Готовая система отслеживания заказов через СДЭК с веб-интерфейсом и API. Система позволяет отслеживать посылки по трек-номерам, отображает историю статусов и предоставляет API для интеграции.

Технологический стек
Backend: Python 3.10, Django 5.2, Django REST Framework

База данных: PostgreSQL

Web-сервер: Nginx + Gunicorn (systemd)

ОС: Ubuntu 22.04 LTS

Дополнительно: BeautifulSoup4, Requests, python-dotenv
